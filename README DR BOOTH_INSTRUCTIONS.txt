______________________________________________________________________
|Welcome to "Primal Bash"                                             |
|_____________________________________________________________________|
|PLAY                                                                 |
|                                                                     |
|1. Run game                                                          |
|                                                                     |
|2. Press "Big Red Go Button" TM.                                     |
|                                                                     |
|3. Let the world procedualy generate                                 |
|	- Press SpaceBar to stop when you want                        |
|	- **When done generated you MUST press SpaceBar**             |
|                                                                     |
|4. Once generated, you can:                                          |
|	- Click different squares to get different types of squares   |
|		BROWN   ==   DIRT                                     |
|		YELLOW  ==   CROPS                                    |
|		GREEN   ==   GRASS                                    |
|		TAN     ==   HUMANS                                   |
|		PINKISH ==   MAMMOTHS                                 |
|		ORANGE  ==   SABERTOOTH TIGERS                        |
|		BLUE    ==   WATER                                    |
|	- Press R to pause and run the game.                          |
|_____________________________________________________________________|
|Gameplay Rules                                                       |
|1. Humans eat crops, and leave grass behind them.                    |
|                                                                     |
|2. Mammoths eat grass and leaves behind dirt.                        |
|   They also can sometimes step on Tigers.                           |
|                                                                     |
|3. Tigers can eat Mammoths, but prefer humans.                       |
